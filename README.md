# LBTextImageShow
文字识别自研解析demo，希望能帮到一部分对于TesseractOCRiOS不是太清楚，或者小白的那一类人

本文是身份证的识别demo, 因为第三方数据库太多, 所以准备删除pod的第三方库, 如果你想下载并运行程序

* 直接pod install将pod所管理的第三方下载下来就可以了
* 将我删掉的语言识别包找回来, 网上有很多资源
